### Installation

1. **Clone and Setup:**
```bash
git clone <repository>
cd ShonaAi-main
pip install -r requirements.txt
```

2. **Install Ollama Model:**
```bash
ollama pull llama3.2:3b
```

3. **Configure Bot:**
```bash
# Windows PowerShell
$env:BOT_TOKEN = "your_telegram_bot_token"

# Linux/Mac
export BOT_TOKEN="your_telegram_bot_token"
```

4. **Run Shona:**
```bash
python main.py
```
